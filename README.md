# Cube Escape & Catch Game

A simple beginner-friendly **C# Windows Forms** game featuring two game modes: **Escape** and **Catch**.  
Perfect for learning basic WinForms, events, and simple game logic.

---

## Description

This project is a lightweight game where the player interacts with a moving cube inside a Windows Forms application.  
Depending on the mode selected, the cube behaves differently and the player has a unique goal.

---

##  Game Modes

### 1. Escape Mode  
In **Escape Mode**, the cube tries to *escape* from an enemy cube.  
- When the cube, the opponent cube jumps to a new random position.  
- The goal is to **catch the cube** even though it keeps running away.

### 2. Catch Mode  
In **Catch Mode**, the player controls one cube using the keyboard.  
- A second cube appears on the screen in random positions.  
- The goal is to **move your cube** and **catch the target cube** by touching it.
